## 🏗️ 系统架构

### 后端技术栈
- **框架**：Django 6.0.2
- **API框架**：Django REST Framework
- **数据库**：MySQL 8.0
- **深度学习**：TensorFlow 2.x
- **模型**：Wide&Deep + DeepFM

### 前端技术栈
- **框架**：Vue.js 3
- **路由**：Vue Router
- **UI组件**：Element Plus
- **构建工具**：Vite
- **HTTP客户端**：Axios
